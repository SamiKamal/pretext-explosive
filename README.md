# Pretext Explosive

Click anywhere to stick a bomb to the text. Watch the fuse tick down. Watch every character scatter like shrapnel with real physics. Watch them reassemble.

All text measurement and layout powered by [@chenglou/pretext](https://github.com/chenglou/pretext) — zero DOM reflows, even during the explosions.

## Run it

```
npm install
npm run dev
```
